# YAML Formatter

Formats and validates YAML locally.

## What it does

- Parses YAML input and emits normalized YAML.
- Validates YAML syntax and reports parser errors.
- Keeps processing entirely on-device.

## Input

- Raw YAML text (or uploaded `.yaml` / `.yml` file).

## Output

- Formatted YAML text.
- Line and character counts.
- Error message when YAML is invalid.
