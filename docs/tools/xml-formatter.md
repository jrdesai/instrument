# XML Formatter

Formats and validates XML locally.

## What it does

- Parses XML input and returns pretty-printed XML.
- Supports configurable indentation (2 or 4 spaces).
- Reports parse/format errors without sending data anywhere.

## Input

- Raw XML text (or uploaded `.xml` file).
- Indent size option.

## Output

- Formatted XML text.
- Line and character counts.
- Error message when XML is invalid.
