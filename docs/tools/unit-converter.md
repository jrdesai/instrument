# Unit Converter

Convert a numeric value from one unit to every other unit in the same category.

**Categories:** data size (decimal and binary prefixes), time, temperature, length, weight, speed, angle (degrees, radians, gradians, turns), and frequency (Hz through THz). Enter a value, choose the source unit, and the tool shows all equivalent values. Month and year use average lengths; Mach uses a fixed sea-level approximation.
