# CSV Previewer

Render CSV or TSV data as a formatted table for quick visual inspection.

## Features

- Supports comma, tab, pipe, and semicolon delimiters
- Toggleable header row (first row used as column names or auto-generated Col 1, Col 2…)
- Row numbers in the first column
- Previews up to 1 000 rows; a warning is shown when the file is larger
- Accepts file upload (drag-and-drop or file picker)
- Accessible from the **CSV ↔ JSON** tool (JSON → CSV direction) and the **Fake Data Generator** (CSV export)
