# Image Converter

Convert images between common formats with local processing only (desktop app).

## Desktop-only

This tool runs in the desktop app only. In the web app, the card is visible as disabled with a "Desktop only" badge.

## Supported input formats

PNG, JPEG, WebP, BMP, TIFF, GIF, ICO, TGA, and PNM.

## Supported output formats

PNG, JPEG, WebP, BMP, TIFF, ICO, TGA, and PNM.

## Operations

- Output format selection
- Quality control for JPEG/WebP
- Resize (width/height, optional aspect lock)
- Rotate (0/90/180/270)
- Flip horizontal / vertical (can combine both)
- Greyscale conversion

## Privacy and metadata

All processing runs locally on your machine. Re-encoding strips EXIF metadata by default.
